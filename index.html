Interactive A* algorithm demonstration in p5.js

By Shardul R.H.


This p5.js program is an interactive visualizer for the A* pathfinding algorithm on a 2D grid. Users can place start and end points, draw or erase obstacles, and observe how the algorithm explores the grid to find the shortest path. It visually distinguishes between explored nodes, the current path, and obstacles, offering a clear real-time view of the algorithmâ€™s decision-making process.

The visualizer supports optional diagonal movement and allows switching between Euclidean and Manhattan distance heuristics. Users can step through the algorithm manually or run it automatically, making it a useful educational tool for understanding how A* works in different conditions.
